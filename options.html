Notifications Personalisées

Lien vers vos notifications personalisées parsées en JSON. Le lien doit contenir "http://" ou "https://".

Les notifications doivent être sous la forme suivante : 

[
	{
		"img": "url_de_l_image",
		"url": "url_de_destination"
		"content": "Contenu en HTML",
	},
	{
		...
	}
]
Cette extension utilise les fonds d'écran de Momentum. Ils sont la propriété de Momentum.
